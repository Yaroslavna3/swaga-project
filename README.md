# LLM-as-a-judge в оценке сгенерированных молекул
Этот проект исследует применение больших языковых моделей в качестве экспертного оценщика качества молекул, полученных генеративными химическими моделями.
В отличие от бинарных метрик вида «валидна / невалидна», система предоставляет осмысленный текстовый фидбек, охватывающий химическую реализуемость, новизну, синтезопригодность и потенциальные риски.
